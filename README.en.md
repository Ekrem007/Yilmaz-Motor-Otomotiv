## ğŸ“¦ Installation

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- Angular CLI

### Steps

1. **Clone the project**
   ```bash
   git clone https://github.com/Ekrem007/Yilmaz-Motor-Otomotiv.git
   cd Yilmaz-Motor-Otomotiv/Frontend/yilmazMotorOtomotiv
   ```

   ```bash
   npm install
   ```

   ```bash
   ng serve
   ```

4. **View the application**
   ```
   http://localhost:4200
   ```
